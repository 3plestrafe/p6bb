HOW TO ADD A STUDENT — 3 STEPS
1.
Enter the email below and click "Add to Whitelist". This lets them log in.
2.
Go to Supabase Auth → Authentication → Users → click "Invite user" or "Add user" and enter the same email.
3.
Set a password for that user in Supabase (click the user → "Send password reset" or set it manually). Give the student their password.
Or set password via SQL Editor in Supabase (replace values):
-- paste in Supabase → SQL Editor UPDATE auth.users SET encrypted_password = crypt('THEIR_PASSWORD', gen_salt('bf')) WHERE email = 'STUDENT_EMAIL'; click to copy
